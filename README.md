# RAG-OCR-System-

A powerful RAG based system use OCR for extracting data from PDF.

This project contains the backend and structure for an OCR and RAG application.

## Structure

- `backend/`: Contains the application logic, API, services, pipelines, etc.
- `frontend/`: Frontend application (placeholder).
- `storage/`: Directory for storing PDFs and OCR results.
- `vector-db/`: Directory for Vector DB data (e.g., Qdrant).
